# Personal Portfolio | Responsive Portfolio Website

## Project Overview

Personal Portfolio is a modern, fully responsive portfolio website designed to showcase my technical skills, projects, education, certifications, and contact information. The website features a clean and professional user interface with smooth navigation, interactive elements, and dark mode support to provide an engaging user experience.

This project demonstrates my front-end development skills using HTML, CSS, and JavaScript while following responsive design principles to ensure seamless performance across desktops, tablets, and mobile devices.

## Installation

Clone the repository:

```bash
git clone https://github.com/PallaviChunchu/personal-portfolio.git
```

Navigate to the project directory:

```bash
cd personal-portfolio
```

Open the project in your preferred code editor.

Run the project by opening the `index.html` file in your preferred web browser.
